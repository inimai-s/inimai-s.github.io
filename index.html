ML Research

In my work as a SuperUROP Landsman Undergraduate Research & Innovation Scholar scholar in the Shavit Lab, I investigate fixed parameter expansion methods for neural networks and associative memory for intelligent document retrieval.

I also performed research at the NASA Jet Propulsion Laboratory (JPL) to build an improved recommendation system that uses a novel combined knowledge graph and GNN architecture to characterize the JPL Problem Reporting System (PRS) database.

Research Projects

Fixed Parameter Sparse Expansion

MIT SuperUROP

Developing new methods to perform fixed parameter expansion on neural networks via neuron duplication and sparsity masks to improve model performance on classification tasks with no increase in compute.

Panza Associative Memory

MIT SuperUROP

Training an embedding model and developing an associative memory module that learns connections between documents beyond semantic similarity to enable natural language retrieval of emails.

Intelligent Knowledge Base Search Tool using LLMs and GNNs

JPL Paper

Built ML recommendation system over LLM-encoded technical anomaly reports from past NASA missions. Utilizes a novel combined knowledge graph (KG) and graph neural net (GNN) architecture and trained over 10,000 reports with link prediction and ensemble training.
